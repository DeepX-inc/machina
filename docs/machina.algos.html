machina.algos package¶

  • This package trains Policy, V function, Q function, etc. by using loss_functional.
  • It is determined here which loss_functional, iterater are used.
  • Also, It is determined how Policy<machina.pols.base.BasePol>, V function, Q function, etc. are updated.
